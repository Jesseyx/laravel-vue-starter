### laravel-vue-starter

学习 Laravel 和 vue 开发的小 demo

> 快速启动

1. `npm install && composer install`

2. `php artisan key:generate`

3. `php artisan migrate`

4. `php artisan db:seed`

5. `php artisan serve`

6. `npm run dev` 

后端使用 laravel，dingo api 和 jwt-auth

前端使用 vuejs，vuex，vue-router 和 vue-resource
